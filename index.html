Here are some information security/programming related resources that are endorsed either by Cyber-Discovery or our community. These have helped us, before and during the Cyber-Discovery events and stages, to practice and train our skills. We hope you find them as useful as we did.

OverTheWire

Wargames offered by the OverTheWire community, which can help you to learn and practice security concepts in the form of fun-filled games.
Visit

Replit

A free, powerful and simple online compiler, IDE, interpreter, and REPL. Replit allows you to Code, compile, and run code in 30+ programming languages.
Visit

SmashTheStack

An ethical hacking environment that supports the simulation of real world software vulnerabilities and allows the use of exploitation techniques.
Visit

Cybrary

Online cyber-security training which you can learn anytime, anywhere with open-source, high quality training from experts, industry professionals and leading companies.
Visit

Udacity

A vast, comprehensive list of free online courses and Nanodegree programmes which range from mastering web design to tech entrepreneurship.
Visit

Codewars

Where programmers achieve code mastery through solving different challenges in different programming languages, with a vibrant active community.
Visit

Codecademy: Learn to code!

An online platform that offers free interactive programming lessons in various different programming languages. Quite a few of us learned Python here.
Visit

Learn RE

Learn the basics of x86 and get hands-on experience with reverse engineering from scratch. Extremely useful for binary reversing CTFs.
Visit

Cyber-Challenge UK

A series of national competitions & programmes, designed to enable more people to become cyber security professionals.
Visit
